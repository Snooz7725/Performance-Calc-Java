# Rate Calculate

Originally Lelushik/TheReDi's; however, it has been improved.

With this mod, you can select an area and calculate:
- Drill rate
- Liquid pump rate
- Factory production & consumption (doesn't include power consumption for now)
- Energy production

## How To Use
### PC
- `C` - hold to select things
- `Ctrl + RMB` - hold to `Ctrl` and press `RMB` to reset the display

### Mobile
- Press left buton to select things
- Press right button to reset the display
